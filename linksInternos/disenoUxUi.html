metodología

Basada en la metodología centrada en el usuario, mi proceso para cualquier desarrollo de aplicación mobile o desktop que comience desde 0, consta de 5 etapas definidas por el método de Design Thinking, las cuales son iterativas, es decir, tienen la flexibilidad de ajustarse a lo largo del proceso.

Etapa 1

Empatizar

  1. Análisis de audiencia
  2. Journey map
  3. Benchmarking
  4. Arquetipos de personas
  5. Problema - Solución

Etapa 2

Definir

  1. Brief de proyecto
  2. Card Sorting
  3. Tree testing
  4. Arq. de información
  5. Userflow basado en flowchart
  6. Wireflow

Etapa 3

Idear

  1. Wireframes
  2. UX Writing
  3. Moodboard
  4. Sistema de diseño · UI Kit

Etapa 4

Prototipar

  1. Prototipos de alta fidelidad
  2. Testeos
  3. Iterar

Etapa 5

Validar

  1. Iterar
  2. Testeos
  3. Lanzamiento
